📖 Hướng Dẫn Sử Dụng Web UI

🚀 Bắt Đầu Nhanh

1. Khởi động Server
python app.py
2. Mở Trình Duyệt
http://localhost:5000
3. Nhập URL Video Bilibili

Dán link video từ: https://www.bilibili.com/video/BV...

4. Chọn Chế Độ & Ngôn Ngữ

Chọn "Chỉ Phụ Đề" hoặc "Lồng Tiếng", sau đó chọn ngôn ngữ gốc

5. Click "Bắt Đầu Dịch"

Chờ kết quả và tải file về

✨ Các Tính Năng

Chỉ Phụ Đề
  • Nhanh (10-20 phút)
  • Tạo file SRT
  • Không cần lồng tiếng
  • Tiết kiệm tài nguyên
Lồng Tiếng
  • Lâu hơn (30-60 phút)
  • Video MP4 hoàn chỉnh
  • Có phụ đề
  • Chất lượng cao

💻 Hướng Dẫn Sử Dụng

📝 Tab "Dịch Video"

Bước 1: Nhập URL video Bilibili (link phải là video công khai)

Bước 2: Chọn ngôn ngữ gốc (Trung, Nhật, Hàn)

Bước 3: Chọn chế độ (Chỉ Phụ Đề hay Lồng Tiếng)

Bước 4: Nếu chọn Lồng Tiếng, chọn phân loại video

Bước 5: Click "Bắt Đầu Dịch" và chờ kết quả

📋 Tab "Danh Sách Công Việc"

Xem toàn bộ lịch sử công việc

Trạng thái:

  • 🔄 Đang Xử Lý - Công việc đang chạy
  • ✅ Hoàn Thành - Công việc thành công
  • ❌ Lỗi - Công việc thất bại

Hành động:

  • Tải kết quả (phụ đề, video, kịch bản)
  • Xem chi tiết công việc

⚙️ Tab "Cài Đặt"

Xem thông tin cấu hình hệ thống

  • Thư mục output
  • Model Whisper sử dụng
  • Chất lượng video mặc định
  • Tỉ lệ mẫu âm thanh

⚙️ Các Chế Độ Chi Tiết

📝 Chế Độ "Chỉ Phụ Đề"

Quá trình:

  1. Tải video từ Bilibili
  2. Trích xuất âm thanh
  3. Nhận diện giọng nói
  4. Dịch sang tiếng Việt
  5. Tạo file phụ đề SRT

Output:

  • subtitle_vi.srt - File phụ đề
  • video_with_subtitle.mp4 - Video có phụ đề
  • transcription.json - Kịch bản

Thời gian: 10-20 phút (tùy độ dài video)

🎙️ Chế Độ "Lồng Tiếng"

Quá trình:

  1. Tải video từ Bilibili
  2. Trích xuất âm thanh
  3. Nhận diện giọng nói
  4. Dịch sang tiếng Việt
  5. Tạo phụ đề
  6. Phát sinh âm thanh lồng tiếng (TTS)
  7. Ghép audio + video + phụ đề

Output:

  • video_vi_dubbed_final.mp4 - Video lồng tiếng hoàn chỉnh
  • subtitle_vi.srt - Phụ đề
  • transcription.json - Kịch bản

Thời gian: 30-60 phút (tùy độ dài video)

🔌 API Endpoints

GET  /api/languages        - Danh sách ngôn ngữ hỗ trợ
GET  /api/categories       - Danh sách phân loại video
POST /api/translate        - Bắt đầu dịch video
GET  /api/job/<job_id>    - Trạng thái công việc
GET  /api/jobs             - Danh sách toàn bộ công việc
GET  /api/download/...     - Tải kết quả
GET  /api/settings         - Cài đặt hệ thống
GET  /api/health           - Health check

🔧 Khắc Phục Sự Cố

❌ Lỗi: "FFmpeg not found"

Giải pháp:

sudo apt-get install ffmpeg  # Ubuntu/Debian
brew install ffmpeg          # macOS
❌ Lỗi: "URL phải là video từ Bilibili"

Giải pháp: Kiểm tra lại URL có chứa "bilibili.com" không

Ví dụ URL hợp lệ:

https://www.bilibili.com/video/BV1Ab4y197mw
https://www.bilibili.com/video/BV17A411N789
❌ Lỗi: "Whisper model download fails"

Giải pháp:

  • Kiểm tra kết nối internet
  • Model được lưu tại: ~/.cache/whisper/
  • Chạy lại script, nó sẽ tạo tiếp từ lần trước
❌ Lỗi: Quá lâu hoặc bị treo

Giải pháp:

  • Chọn video ngắn (dưới 10 phút) để test
  • Dùng chế độ "Chỉ Phụ Đề" (nhanh hơn)
  • Kiểm tra bộ nhớ RAM (cần ít nhất 4GB)
  • Kiểm tra dung lượng ổ cứng (cần ít nhất 10GB)
